Scripts will NOT be reviewed, so if your script gets reported any future request will be ignored.

Before requesting any Roblox script it must meet these requirements:

  • Script has to be loadingstring or raw code
  • Script must be tested by the publisher.
  • Script must be undetected.
  • Script should not lead to account bans or in-game bans.
  • Script should not be buggy and unusable
  • Script should not be NSFW

If your request does not meet these requirements it'll be taken down and any future request will be ignored.
